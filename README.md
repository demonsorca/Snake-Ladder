# Snake Ladder Game
I have implemented snake ladder game using Pyhton and Pygame.

## Steps to Run Game
- Download the repo.
- Open the exracted folder in pycharm.
- Add python interpreter.
- Install pygame module.
- Run ChessMain.py and Enjoy!!
